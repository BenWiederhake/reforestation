# reforestation

reforestation is a silly, fun project about "reforestation". It makes a green wall, while implementing new and exciting features automatically.

- [TODOs](#todos)

## Install

Call `keep_up.py` every now and then, for example via cron script.

## Usage

Choose any of:
- `./keep_up.py` in order to generate all missing commits
- `./reforest.py && git add reforest.py && faketime "2012-12-12 12:12:12" git commit -m "Clever message"` to manually generate a single commit
